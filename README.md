# image-calibration
We perform a task called camera calibration to correct distortion of images.
The image used in the practice is a chessboard video.
Image calibration is performed by photographing the chessboard from various angles. 
An image in which distortion is corrected as a result of execution can be obtained.
Information on the lens' focal length, center point, and convexity of the lens is included in the calib_data.npz file. 
This is one of the key keys to performing AR work that allows me to draw the shapes you want on the chessboard.
The result video is in the calibration_record.mp4.
